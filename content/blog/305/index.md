## 安装

官方安装文档教程：
- [Installing ZSH](https://github.com/ohmyzsh/ohmyzsh/wiki/Installing-ZSH)
- [Install oh-my-zsh now](https://ohmyz.sh/#install)

我在`Ubuntu`下执行以下命令：

``` bash
apt install zsh
sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```
